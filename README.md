# CenterAlignActionBar

----
## 原生的問題
> Android 原生的 ToolBar 是靠左對齊，如果放上icon 很容易有title 超過寬度, 或是蓋過icon的問題.

----
## 我能做到

1. 不被遮蔽
2. 置中對齊
3. 包裹任意元件
